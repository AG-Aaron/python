# python
80/20 rule python 

•	Data Types & Variables Variables, strings, integers, floats, booleans, None — the absolute basics you use in every program.
•	Data Structures Lists, dictionaries, tuples, and sets. Dictionaries alone power a huge portion of real Python code.
•	Control Flow if/elif/else, for loops, while loops, break/continue. Nearly all logic reduces to these.
•	Functions Defining functions, parameters, return values, and default arguments. This is where reusable code begins.
•	List Comprehensions [x for x in items if condition] — Pythonic, fast, and everywhere in real codebases.
•	File I/O Reading and writing files with open(). Almost every useful script touches the filesystem.
•	Error Handling try/except blocks. Production code lives and dies by this.
•	Modules & Imports Using import and the standard library (os, sys, json, datetime). You rarely write everything from scratch.
•	Classes (Basic OOP) Understanding class, __init__, and methods.
•	String Manipulation .split(), .join(), .strip(), f-strings, .replace()
